# 🎮 Games Arena — Gaming Landing Page

A modern and responsive **gaming landing page** built using **HTML5**, **CSS3**, and **Bootstrap 5** as part of my Front-End Development journey.

This project focuses on building professional layouts using Bootstrap, creating interactive UI elements, and delivering a smooth user experience across all devices.

## 📌 Features

✅ Fully responsive design across all screen sizes  
✅ Modern gaming UI with dark theme  
✅ Responsive navbar with smooth navigation  
✅ Interactive carousel for latest games  
✅ Team section with advanced hover effects  
✅ Smooth animations and transitions  
✅ Contact form with custom styling  
✅ Clean and structured layout using Bootstrap

## 🛠️ Tech Stack

**HTML5** — page structure  
**CSS3** — custom styling and animations  
**Bootstrap 5** — grid system and UI components  
**Flexbox** — layout enhancements  
**Media Queries** — responsive design  
**CSS Variables** — consistent styling
